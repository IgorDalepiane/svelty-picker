Custom Element

Custom element is defined as extension to native input tag. Custom element supports all options that can be passed as string (So all except initialDate).

Picker initialized as custom element: 
<script>
  import { registerElement } from 'svelty-picker';

  registerElement('el-picker');
</script>

<!-- somewhere in HTML -->
<input type="text" is="el-picker">

Custom element supports 2 custom from and to properties, which should specify id of different datetime picker for limiting startDate or endDate. Check source how it is defined.
